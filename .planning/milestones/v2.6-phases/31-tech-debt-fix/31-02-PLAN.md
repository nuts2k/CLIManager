    **Rust 后端 — src-tauri/src/commands/traffic.rs:**

    将 3 个 Tauri command 函数签名从 `tauri::State<'_, TrafficDb>` 改为 `app_handle: tauri::AppHandle`，使用 `try_state` 安全获取 DB 引用：

    ```rust
    #[tauri::command]
    pub async fn get_recent_logs(
        app_handle: tauri::AppHandle,
        limit: Option<i64>,
    ) -> Result<Vec<crate::traffic::log::TrafficLogPayload>, String> {
        use tauri::Manager;
        let db = app_handle
            .try_state::<crate::traffic::TrafficDb>()
            .ok_or_else(|| "数据库不可用（DB 初始化失败，请检查磁盘空间和权限）".to_string())?;
        let limit = limit.unwrap_or(100).min(1000);
        db.query_recent_logs(limit).map_err(|e| e.to_string())
    }
    ```

    对 `get_provider_stats` 和 `get_time_trend` 同理修改：
    - 参数列表去掉 `traffic_db: tauri::State<'_, crate::traffic::TrafficDb>`，换为 `app_handle: tauri::AppHandle`
    - 函数体开头加 `use tauri::Manager;` 和 `let db = app_handle.try_state::<crate::traffic::TrafficDb>().ok_or_else(|| "数据库不可用（DB 初始化失败，请检查磁盘空间和权限）".to_string())?;`
