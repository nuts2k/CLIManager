    --- clean_schema() ---
    - 移除顶层 format 字段：{"type":"string","format":"uri"} => {"type":"string"}
    - 递归移除 properties 内嵌套的 format 字段
    - 递归移除 items 内嵌套的 format 字段
    - 无 format 字段时原样返回
    - 移除 default 字段（OpenAI 不兼容）

    --- anthropic_to_openai() ---
    - system 字符串 => messages[0] role:system
    - system 数组（多条 text block）=> 多条 system messages
    - text content block => 单条 user/assistant message content 字符串
    - 多个 text content blocks => 用换行合并
    - tool_use content block => assistant message 的 tool_calls 数组
    - tool_result content block => role:tool message（含 tool_call_id 和 content 字符串）
    - tool_result content 为数组时序列化为字符串
    - image content block（base64）=> image_url { url: "data:{media_type};base64,{data}" }
    - 工具定义转换：input_schema => function.parameters + type:"function" 包装
    - BatchTool 类型工具被过滤（不出现在转换结果中）
    - clean_schema 应用到每个工具的 function.parameters
    - cache_control 透传：system message 保留 cache_control、text content 保留、tool 定义保留
    - stop_sequences => stop、max_tokens 透传、temperature 透传、top_p 透传、stream 透传
    - thinking blocks（type:"thinking"）静默丢弃
    - 未知 content block type 静默丢弃
    - model 字段原样透传（Phase 14 不做映射，Phase 15 handler 层处理）
  </behavior>
