实现 Claude overlay 专用的“config 目录”解析与读写函数，遵循项目既有 iCloud fallback 规则：

1) 在 src-tauri/src/storage/icloud.rs 新增：
- `pub enum StorageLocation { ICloud, LocalFallback }`（用于 UI 感知）
- `pub struct OverlayStorageInfo { location: StorageLocation, dir: PathBuf, file_path: PathBuf, sync_enabled: bool }`
- `pub fn get_icloud_config_dir() -> Result<(PathBuf, StorageLocation), AppError>`
  - iCloud 可用：`~/Library/Mobile Documents/com~apple~CloudDocs/CLIManager/config`
  - iCloud 不可用：warn log + fallback `~/.cli-manager/config`
- `pub fn get_claude_overlay_path() -> Result<(PathBuf, StorageLocation), AppError>`
  - 文件名固定：`claude-settings-overlay.json`
- `pub fn read_claude_settings_overlay() -> Result<(Option<String>, OverlayStorageInfo), AppError>`
  - 文件不存在：返回 Ok((None, info))（视为 noop）
  - 文件存在：返回原始文本内容（不在此处做 JSON 校验；校验由 command/apply 执行）
- `pub fn write_claude_settings_overlay(content: &str) -> Result<OverlayStorageInfo, AppError>`
  - 使用现有 `atomic_write()` 写入

2) 注意：不要影响现有 providers 目录逻辑；新增函数保持独立。
3) info 中的 `sync_enabled`：ICloud=true，LocalFallback=false。

1) 新建 `src-tauri/src/commands/claude_settings.rs`，实现 2 个 Tauri commands：

- `#[tauri::command] pub fn get_claude_settings_overlay() -> Result<GetClaudeSettingsOverlayResponse, AppError>`
  - 读取 overlay 内容（Option<String>）+ OverlayStorageInfo（含 location + path + sync_enabled）

- `#[tauri::command] pub fn set_claude_settings_overlay(app_handle: tauri::AppHandle, overlay_json: String) -> Result<SetClaudeSettingsOverlayResponse, AppError>`
  - 仅做 **JSON 校验**：必须是合法 JSON 且 root 为 object；否则 Err(AppError::Validation(...))
  - 将 overlay_json 使用 serde_json 解析后，再用 `to_string_pretty` 规范化格式写入（避免乱格式导致 diff 抖动）
  - 写入前通过 `SelfWriteTracker` 记录 overlay 文件路径，避免后续 watcher 扩展时处理自写事件：
    - `let tracker = app_handle.state::<crate::watcher::SelfWriteTracker>();`
    - `tracker.record_write(overlay_path.clone());`

2) 将该模块加入 `src-tauri/src/commands/mod.rs`：`pub mod claude_settings;`

3) 在 `src-tauri/src/lib.rs` 的 invoke_handler 中注册：
- `commands::claude_settings::get_claude_settings_overlay`
- `commands::claude_settings::set_claude_settings_overlay`

注意：本 Plan **不实现 apply**，也不要在 set 命令里隐式修改 `~/.claude/settings.json`。
