1. **创建 `src/components/updater/useUpdater.ts`（UPD-01、UPD-02、UPD-04）：**

   导出 `useUpdater` hook，管理更新检查和下载状态：

   ```typescript
   // 状态类型
   type UpdateStatus = 'idle' | 'checking' | 'available' | 'downloading' | 'ready' | 'error';

   interface UseUpdaterReturn {
     status: UpdateStatus;
     newVersion: string | null;       // 新版本号
     currentVersion: string;          // 当前版本（从 Cargo.toml 读取的版本）
     progress: number;                // 下载进度 0-100
     error: string | null;
     checkForUpdate: () => Promise<void>;
     downloadAndInstall: () => Promise<void>;
     dismissUpdate: () => void;       // 稍后提醒
   }
   ```

   实现要点：
   - 使用 `import { check } from '@tauri-apps/plugin-updater'` 和 `import { relaunch } from '@tauri-apps/plugin-process'`
   - `currentVersion` 通过 `import { getVersion } from '@tauri-apps/api/app'` 获取
   - `checkForUpdate()`：调用 `check()`，成功时设置 `status='available'` 和 `newVersion`，无更新设 `status='idle'`
   - `downloadAndInstall()`：调用 `update.downloadAndInstall(callback)`，在 callback 中根据 DownloadEvent 更新 `progress`（0-100）。Started 时 contentLength 可能为 undefined，此时 progress 设为 -1 表示不确定态。Finished 后设 `status='ready'`，然后调用 `relaunch()`
   - `dismissUpdate()`：使用 `useRef` 标记 `dismissedThisSession`（用户决策：本次启动不再弹窗），设 `status='idle'`
   - 错误处理：catch 所有异常，设 `status='error'`、`error` 消息。更新检查失败静默处理（不阻断主流程）
   - 注意：开发模式下（`window.__TAURI_INTERNALS__` 不存在或 dev server）check() 会抛异常，需 catch 静默

2. **创建 `src/components/updater/UpdateDialog.tsx`（UPD-03）：**

   复用项目已有的 `Dialog` 组件（用户决策：模态对话框形式，不用 Toast）：

   ```typescript

     open: boolean;
     onOpenChange: (open: boolean) => void;
     status: UpdateStatus;
     currentVersion: string;
     newVersion: string | null;
     progress: number;
     onUpdate: () => void;      // 开始下载安装
     onRemindLater: () => void; // 稍后提醒
   }
   ```

   UI 结构：
   - **状态 `available`**：DialogHeader 显示标题（如"发现新版本"），DialogDescription 显示当前版本和新版本号（用户决策：只显示版本号，不展示更新日志）。DialogFooter 两个按钮："稍后提醒"（outline）和"立即更新"（default）
   - **状态 `downloading`**：显示下载中文案 + 进度条。progress >= 0 时显示百分比进度条（使用 Tailwind 实现简单进度条 div，不用额外 UI 库），progress === -1 时显示不确定态（animate-pulse 动画条）。此状态下不可关闭对话框（`showCloseButton={false}`）
   - **状态 `ready`**：显示"安装完成，即将重启..."
   - **状态 `error`**：显示错误信息 + 关闭按钮
   - 所有文案使用 i18n key（`updater.*`）
   - 进度条样式：使用 Tailwind 的 `h-2 rounded-full bg-primary` 在 `bg-muted` 容器中实现
