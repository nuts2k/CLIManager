  <action>
在 src/index.css 的 `:root` 块中，在 Plan 01 新增的品牌色变量之后，添加间距阶梯 CSS 自定义属性和圆角规范的文档注释。

新增间距变量（在 :root 块内）：
```css
/* 间距阶梯 (4px 基数) */
--space-xs: 0.25rem;   /* 4px  — 元素内紧凑间隙 */
--space-sm: 0.5rem;    /* 8px  — label 与 input 间距、图标间隙 */
--space-md: 0.75rem;   /* 12px — 同组元素间距 */
--space-lg: 1rem;      /* 16px — 区块内 padding、卡片内 padding */
--space-xl: 1.5rem;    /* 24px — 区块间距、页面主 padding */
--space-2xl: 2rem;     /* 32px — 页面级大间距 */
```

在 index.css 文件顶部（@import 之后、@custom-variant 之前）添加设计规范注释块：
```css
/*
 * ===== 设计规范 =====
 *
 * 间距阶梯 (基于 4px)：
 *   xs=4px(1)  sm=8px(2)  md=12px(3)  lg=16px(4)  xl=24px(6)  2xl=32px(8)
 *   括号内为 Tailwind spacing 单位
 *
 * 使用场景：
 *   - label 与 input 间距: gap-1.5 (6px，允许的中间值)
 *   - 同组元素间距: gap-2(8px) 或 gap-3(12px)
 *   - 卡片内 padding: p-3(12px) 或 p-4(16px)
 *   - 区块间距: space-y-6(24px)
 *   - 页面 padding: p-4(16px) 或 p-6(24px)
 *
 * 圆角规范：
 *   - 卡片/对话框外框: rounded-lg (--radius-lg, 10px)
 *   - 按钮/输入框: rounded-md (--radius-md, 8px)
 *   - 小型指示物: rounded-full
 *   - Tab 容器: rounded-lg
 *
 * 颜色使用：
 *   - 品牌强调: brand-accent (橙 #F97316)
 *   - 活跃状态: status-active
 *   - 成功/在线: status-success
 *   - 警告: status-warning
 *   - 危险/删除: destructive (shadcn 内置)
 */
```

注意：不要修改 Plan 01 已添加的变量。只追加间距变量和注释。
