    **步骤 1 — 备份旧图标：**
    创建 src-tauri/icons-backup/ 目录，将 src-tauri/icons/ 下所有现有图标文件（不含 tray/ 子目录）复制进去保留。

    **步骤 2 — 生成应用图标 SVG：**
    用 Write 工具创建 src-tauri/icons/app-icon.svg，设计要求：
    - 画布 1024x1024（方便缩放到各尺寸）
    - 深色背景填满整个画布（深灰/近黑色，如 #1A1A2E 或 #111827，与应用暗色主题呼应）
    - 中心圆形枢纽节点：品牌橙色 #F97316，大小约画布 15-18%，居中
    - 4-6 个外围小圆点节点：同为品牌橙色，大小约中心节点的 35-50%
    - 外围节点分布不均匀（非等角间距），到圆心的距离也有变化（约画布半径的 55-75%），营造有机自然连接感
    - 中心到各外围节点的连接线：橙色，宽度约 8-12px，可用直线或微弧线
    - 注意所有元素需在画布内留足边距（至少 10% padding），macOS 会自动裁剪为 squircle

    **步骤 3 — 从 SVG 生成 512px 主 PNG：**
    使用 sips 或 qlmanage 将 SVG 转换为 1024x1024 PNG，然后缩放到 512x512 作为 icon.png。
    具体命令流程：
    ```bash
    # 方案 A：使用 qlmanage 渲染 SVG 为 PNG
    qlmanage -t -s 1024 -o /tmp/ src-tauri/icons/app-icon.svg
    # 输出文件为 /tmp/app-icon.svg.png，重命名并缩放
    sips -z 512 512 /tmp/app-icon.svg.png --out src-tauri/icons/icon.png

    # 方案 B：若 qlmanage 效果不佳，使用 Python + cairosvg（需 pip install cairosvg）
    # python3 -c "import cairosvg; cairosvg.svg2png(url='src-tauri/icons/app-icon.svg', write_to='src-tauri/icons/icon.png', output_width=512, output_height=512)"
    ```

    **步骤 4 — 生成全套尺寸 PNG：**
    从 512px icon.png 用 sips 缩放生成各尺寸：
    ```bash
    sips -z 32 32 src-tauri/icons/icon.png --out src-tauri/icons/32x32.png
    sips -z 128 128 src-tauri/icons/icon.png --out src-tauri/icons/128x128.png
    sips -z 256 256 src-tauri/icons/icon.png --out src-tauri/icons/128x128@2x.png
    ```

    Windows Square 系列：
    ```bash
    sips -z 30 30 src-tauri/icons/icon.png --out src-tauri/icons/Square30x30Logo.png
    sips -z 44 44 src-tauri/icons/icon.png --out src-tauri/icons/Square44x44Logo.png
    sips -z 71 71 src-tauri/icons/icon.png --out src-tauri/icons/Square71x71Logo.png
    sips -z 89 89 src-tauri/icons/icon.png --out src-tauri/icons/Square89x89Logo.png
    sips -z 107 107 src-tauri/icons/icon.png --out src-tauri/icons/Square107x107Logo.png
    sips -z 142 142 src-tauri/icons/icon.png --out src-tauri/icons/Square142x142Logo.png
    sips -z 150 150 src-tauri/icons/icon.png --out src-tauri/icons/Square150x150Logo.png
    sips -z 284 284 src-tauri/icons/icon.png --out src-tauri/icons/Square284x284Logo.png
    sips -z 310 310 src-tauri/icons/icon.png --out src-tauri/icons/Square310x310Logo.png
    ```

    StoreLogo：
    ```bash
    sips -z 50 50 src-tauri/icons/icon.png --out src-tauri/icons/StoreLogo.png
    ```

    **步骤 5 — 生成 icns：**
    使用 iconutil 从 iconset 文件夹生成 .icns：
    ```bash
    mkdir -p /tmp/AppIcon.iconset
    sips -z 16 16 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_16x16.png
    sips -z 32 32 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_16x16@2x.png
    sips -z 32 32 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_32x32.png
    sips -z 64 64 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_32x32@2x.png
    sips -z 128 128 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_128x128.png
    sips -z 256 256 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_128x128@2x.png
    sips -z 256 256 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_256x256.png
    sips -z 512 512 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_256x256@2x.png
    sips -z 512 512 src-tauri/icons/icon.png --out /tmp/AppIcon.iconset/icon_512x512.png
    # 对于 512@2x 需要 1024px 源
    sips -z 1024 1024 /tmp/app-icon-1024.png --out /tmp/AppIcon.iconset/icon_512x512@2x.png
    iconutil -c icns /tmp/AppIcon.iconset -o src-tauri/icons/icon.icns
    ```
    注意：512x512@2x 需要 1024px 源图像，在步骤 3 中保留 1024px 中间文件。

    **步骤 6 — 生成 ico：**
    使用 sips 生成多尺寸 PNG，然后用 Python 的 Pillow 合并为 ico：
    ```bash
    python3 -c "
    from PIL import Image
    img = Image.open('src-tauri/icons/icon.png')
    sizes = [(16,16),(24,24),(32,32),(48,48),(64,64),(128,128),(256,256)]
    img.save('src-tauri/icons/icon.ico', format='ICO', sizes=sizes)
    "
    ```
    若 Pillow 不可用，尝试 `pip3 install Pillow` 安装，或使用 ImageMagick `convert` 命令作为备选。
