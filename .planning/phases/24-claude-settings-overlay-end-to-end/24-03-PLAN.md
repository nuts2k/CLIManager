  <name>Task 1: 新增 adapter/json_merge.rs 实现深度合并 + null 删除 + 保护字段剥离</name>
  <files>src-tauri/src/adapter/json_merge.rs, src-tauri/src/adapter/mod.rs</files>
  <behavior>
- 合并规则：
  - base/object + overlay/object => 递归合并每个 key
  - base/any + overlay/array => overlay 数组整体替换
  - base/any + overlay/scalar => overlay 覆盖
  - overlay 的某个 key 为 null => 从 base object 删除该 key（如果不存在则 noop）
- 保护字段剥离：
  - 给定 overlay Value 与保护路径集合（本阶段固定为 env.ANTHROPIC_AUTH_TOKEN / env.ANTHROPIC_BASE_URL）
  - overlay 中出现上述字段：应从 overlay 中移除（忽略），同时返回被移除的路径列表供 UI 提示（后续 plan 会用）
- 合并函数不应在 root 非 object 时 panic：返回 AppError::Validation
  </behavior>
  <action>
1) 新建 `src-tauri/src/adapter/json_merge.rs`，实现以下 API（尽量小而清晰，便于 Phase 25 测试扩展）：

- `pub const PROTECTED_ENV_KEYS: [&str; 2] = ["ANTHROPIC_AUTH_TOKEN", "ANTHROPIC_BASE_URL"];`
- `pub struct StripResult { pub stripped_paths: Vec<String>, pub overlay: serde_json::Value }`
- `pub fn strip_protected_fields(overlay: &serde_json::Value) -> Result<StripResult, AppError>`
  - 仅处理 overlay root object；否则 Validation
  - 若 overlay.env 不是 object：不做深挖（保持 overlay 原样），但如果能明确找到保护字段则移除
  - stripped_paths 用 "env.ANTHROPIC_AUTH_TOKEN" 形式
- `pub fn merge_with_null_delete(base: &mut serde_json::Value, overlay: &serde_json::Value) -> Result<(), AppError>`
  - 要求 base 为 object；否则 Validation

- 若 overlay 存在：
  - 先 strip 掉 overlay 中的保护字段
  - 再将 overlay 深度合并进 root（含 null 删除）
  - 最后再次写回保护字段（保证优先级）
- overlay 不存在：行为应与当前完全一致
  </behavior>
  <action>
在 `src-tauri/src/adapter/claude.rs` 中升级 `patch_claude_json()` 逻辑，但必须保持“surgical patch”核心价值：不要无关字段重写为默认值。

实现策略（允许用 serde_json::Value 操作，但避免改变无关 key 的形态）：
1) 在 patch() 内，读取 existing JSON 为 Value（当前已有预校验）。
2) 对 root 执行：
   - 先确保 env object 存在（现有逻辑保留）
   - 从 storage 读取 overlay（调用你在 Plan 24-01 中新增的 storage 函数；若 executor 为减少耦合，也可在此阶段先只支持通过参数注入 overlay，后续 Plan 再接入——但最终本 Phase 需要 end-to-end，所以建议直接读取 storage）
   - 若 overlay 存在：解析为 Value，必须是 object，否则返回 Validation（对应 COVL-12 会在 apply 命令里更显式处理，但 adapter 层也应防御）
   - 调用 `strip_protected_fields()` 得到 stripped overlay
   - 调用 `merge_with_null_delete(&mut root, &overlay)`
3) 无论 overlay 是否存在，都要在最后写回：
   - env.ANTHROPIC_AUTH_TOKEN = provider.api_key
   - env.ANTHROPIC_BASE_URL = provider.base_url
4) pretty 输出（serde_json::to_string_pretty）保持现有格式。

注意：本 Task 不需要在 UI 上显示 stripped_paths；只需保证保护字段最终优先。
