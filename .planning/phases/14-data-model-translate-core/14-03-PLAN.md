    - "length" => "max_tokens"
    - "tool_calls" => "tool_use"
    - "function_call" => "tool_use"
    - "content_filter" => "end_turn"
    - "" (空) => "end_turn"
    - "unknown_reason" (未知) => "end_turn"

    --- openai_to_anthropic() ---
    基础文本响应 (RESP-01):
    - choices[0].message.content 为字符串 => 单个 text content block
    - choices[0].message.content 为 null + refusal 字段 => 使用 refusal 作为 text
    - choices[0].message.content 为空字符串 => 空 content 数组
    - 响应包含 model 字段 => 透传到 Anthropic 响应

    工具调用响应 (RESP-02):
    - choices[0].message.tool_calls 数组 => 每个转换为 tool_use content block
    - tool_call.function.arguments（字符串）=> 反序列化为 JSON 对象赋给 input
    - tool_call.function.arguments 反序列化失败 => 包装为 {"raw": "原字符串"}
    - tool_calls 和 content 同时存在 => text block + tool_use blocks

    finish_reason 映射 (RESP-03):
    - choices[0].finish_reason => stop_reason（使用 map_finish_reason()）

    usage 映射 (RESP-04):
    - prompt_tokens => input_tokens
    - completion_tokens => output_tokens
    - prompt_tokens_details.cached_tokens => cache_read_input_tokens（如果存在）
    - usage 为 null 时 => 提供默认值 0

    整体输出格式:
    - 返回完整 Anthropic 响应结构：id, type:"message", role:"assistant", model, content, stop_reason, stop_sequence:null, usage
    - id 由 OpenAI 响应的 id 字段透传（前缀 "msg_" 如果原 id 不含此前缀）
  </behavior>
